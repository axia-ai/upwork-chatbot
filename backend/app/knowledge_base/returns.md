# Returns & Exchanges

North Star Outfitters offers a **30-day return policy**. Items must be **unused**
and in their **original packaging** to qualify for a return or exchange.

To start a return or exchange, visit our returns portal:
https://northstar.example.com/returns

## How returns work
- You have 30 days from the delivery date to request a return.
- The item must be unused, with tags attached, in its original packaging.
- Once we receive and inspect the item, your refund is issued to the original
  payment method.
- Exchanges follow the same rules — start one through the returns portal and pick
  the replacement size or item.

## Common questions
- **A jacket runs small — can I exchange it?** Yes. As long as it is unused and
  in its original packaging within 30 days, start an exchange at the returns
  portal and select the new size.
- **My item arrived damaged.** We're sorry about that. Damaged-on-arrival items
  are covered — start a return at the portal, and if you'd like a hand, ask to
  speak with a live agent.
